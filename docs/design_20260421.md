# UI Implementation Guide: Vibrant Candy Style (with Original Logo)

이 문서는 기존 서비스의 브랜드 정체성(로고)을 유지하면서, UI 시스템을 'Vibrant Candy' 스타일로 현대화하기 위한 개발 명세서입니다.

## 1. Brand Asset (Preserved)

### Logo Image
- **Asset:** 기존 캡처본의 로고 이미지 (`{{DATA:IMAGE:IMAGE_4}}` 참조)
- **Implementation:** 상단 영역에 기존 텍스트 형태의 로고 이미지를 그대로 배치합니다.
- **Placement:** 카드 상단 중앙 (상단 여백 40px 유지)

---

## 2. New Design Tokens (Vibrant Candy)

### Colors
- **Primary (Blue):** `#0055c4` (로그인 버튼 및 활성 상태)
- **Primary Light:** `#739eff` (포커스 상태 및 악센트)
- **Background:** `#ffffff` (메인 로그인 카드 배경)
- **Surface (Input):** `#f0f2ff` (입력창 배경색)
- **Text Primary:** `#252c51` (제목 및 주요 텍스트)
- **Text Secondary:** `#535981` (라벨 및 도움말 텍스트)
- **Page Background Gradient:** `linear-gradient(to bottom right, #ffffff, #f0f2ff)`

### Typography
- **Primary Font:** `Lexend`, sans-serif (기존 로고 제외한 모든 UI 텍스트에 적용)
- **Labels:** Font-weight 500 (Medium), Letter-spacing tight
- **Button Text:** Font-weight 700 (Bold)

### Layout & Shapes
- **Main Card Border Radius:** `2.5rem` (40px)
- **Input & Button Border Radius:** `1rem` (16px) 또는 Full Round
- **Card Shadow:** `0px 12px 32px rgba(37, 44, 81, 0.06)`

---

## 3. Component Specifications

### Input Fields (User ID / Password)
- **Background:** `#f0f2ff`
- **Border:** None (또는 포커스 시 `#739eff` 1.5px Outline)
- **Padding:** `1rem 1.25rem`
- **Icons:** Material Icons 스타일, 컬러 `#535981`

### Primary Button (Sign In)
- **Background:** `#0055c4`
- **Text Color:** `#ffffff`
- **Shadow:** `0px 8px 20px rgba(0, 85, 196, 0.2)`
- **Interaction:** `active:scale-95`, `transition: all 0.2s cubic-bezier(0.34, 1.56, 0.64, 1)`

---

## 4. Development Notes
- 로고는 텍스트 폰트가 아닌 **이미지 에셋**으로 처리하여 고유한 디자인을 유지합니다.
- 나머지 모든 폼 요소와 레이아웃은 위 명시된 `Vibrant Candy` 토큰을 따릅니다.
- 상세 코드는 Stitch 툴바의 **'</> View Code'**를 통해 익스포트할 수 있습니다.
