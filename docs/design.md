# 📝 돈 세는 가계부 통합 디자인 및 구현 명세서 (ver 1.1)

본 문서는 '한글 캘린더 가계부 작업창', '다중 거래 내역 입력 팝업', '상세 거래 입력(스프레드시트 뷰)' 디자인을 기반으로 하며, 풀스택 개발자가 프론트엔드 UI 및 백엔드 연동을 구현하기 위한 상세 사양을 포함합니다.

---

## 1. 디자인 시스템 (Design System)

- **폰트 (Typography)**: `Manrope` (또는 Pretendard). 가독성과 세련된 느낌을 주는 산세리프 폰트 적용.
- **색상 체계 (Color Scheme)**:
  - **수입 (Income)**: `#FF3B30` (빨간색) - 강조 및 수입 데이터 표현
  - **지출 (Expense)**: `#007AFF` (파란색) - 강조 및 지출 데이터 표현
  - **Background**: `#F8F9FA` (연한 그레이 배경)
  - **Card/Sheet**: `#FFFFFF` (순백색)
  - **Text (Main)**: `#1A1A1A`
  - **Text (Secondary)**: `#8E8E93`

## 2. 화면별 상세 구현 사양

### A. 한글 캘린더 가계부 작업창 ({{DATA:SCREEN:SCREEN_26}} 기반)
- **명칭**: 돈 세는 가계부 - 캘린더 뷰
- **레이아웃**: 7열 그리드 방식의 월간 캘린더.
- **데이터 바인딩**:
  - 각 날짜별 상단에 **수입(빨간색)** 총액, 하단에 **지출(파란색)** 총액 표시.
  - 당일은 하이라이트(파란색 테두리 등) 처리.
- **하단 요약**: 해당 월의 총 수입, 총 지출, 잔여 예산을 실시간으로 집계하여 표시.
- **인터랙션**:
  - 날짜 클릭 시 해당 날짜의 '다중 입력 팝업' 노출.
  - 하단 중앙 '작성하기(+)' 버튼 클릭 시 오늘 날짜 기준 입력 팝업 노출.

### B. 다중 거래 내역 입력 팝업 ({{DATA:SCREEN:SCREEN_32}} 기반)
- **컴포넌트**: 하단 시트(Bottom Sheet) 또는 중앙 모달 형태.
- **기능**:
  - **항목 리스트**: 현재 팝업 내에서 추가된 거래 내역들을 상단에 카드로 표시.
  - **상세 추가 버튼**: 상단 우측에 배치. 클릭 시 '상세 거래 입력(스프레드시트 뷰)' 화면으로 전환.
  - **입력 폼**: 수입/지출 토글, 금액(천 단위 콤마), 카테고리(아이콘 선택형), 사용처 입력 필드.
  - **항목 추가**: 입력된 정보를 임시 리스트에 추가하고 폼을 초기화.
  - **저장하기**: 리스트에 담긴 모든 내역을 서버에 한꺼번에 전송.

### C. 상세 거래 입력 - 스프레드시트 뷰 ({{DATA:SCREEN:SCREEN_30}} 기반)
- **UI 구조**: 엑셀/구글 시트 방식의 데이터 그리드.
- **기능**:
  - **인라인 편집**: 각 셀(Cell) 클릭 시 즉시 텍스트/날짜 입력 및 카테고리 선택 가능.
  - **행 추가/삭제**: 하단 또는 우측 버튼을 통해 빠르게 행(Row)을 생성.
  - **대량 저장**: 우측 상단 '완료' 버튼 클릭 시 모든 행의 데이터를 일괄 검증 후 저장.
- **반응형**: 모바일에서도 가로 스크롤을 통해 모든 열(Column)에 접근 가능하도록 최적화.

## 3. 개발 가이드 및 API 제안

### 프론트엔드 (Frontend)
- **State 관리**: 입력 팝업의 임시 리스트와 스프레드시트의 행 데이터를 관리하기 위한 `Context` 또는 `State Library` 사용 권장.
- **컴포넌트**: 캘린더 라이브러리(FullCalendar 등) 커스터마이징 또는 직접 구현.

### 백엔드 API (Backend)
1. **GET** `/api/v1/calendar?year=2026&month=04`: 일자별 수입/지출 합계 데이터 호출.
2. **POST** `/api/v1/transactions/bulk`: 다중 거래 내역을 배열 형태로 전송.
   ```json
   {
     "date": "2026-04-15",
     "items": [
       {"type": "income", "amount": 15000, "category": "기타", "merchant": "상여금"},
       {"type": "expense", "amount": 21800, "category": "식비", "merchant": "스타브스"}
     ]
   }
   ```
